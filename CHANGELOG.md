# Changelog

Все значимые изменения в проекте Document QA Agent будут документированы в этом файле.

## [1.0.0] - 2025-01-17

### Добавлено

#### Основные возможности
- RAG (Retrieval-Augmented Generation) система для ответов на вопросы по документам
- Парсинг документов в форматах PDF, DOCX, Markdown, TXT
- Извлечение таблиц из документов
- Семантический поиск с использованием векторных эмбеддингов
- Интеграция с YandexGPT и GigaChat
- Веб-интерфейс на базе Streamlit
- Поддержка русского языка

#### Компоненты
- `src/parsers/` - модули парсинга документов
  - `BaseParser` - базовый класс для парсеров
  - `PDFParser` - парсер PDF документов
  - `DOCXParser` - парсер Word документов
  - `TextParser` - парсер текстовых файлов и Markdown

- `src/embeddings/` - модули работы с эмбеддингами
  - `Embedder` - генерация эмбеддингов с использованием Sentence Transformers
  - `TextSplitter` - разбиение текста на чанки
  - `VectorStore` - векторное хранилище на базе ChromaDB

- `src/rag/` - RAG pipeline
  - `LLMClient` - базовый класс для LLM клиентов
  - `YandexGPTClient` - клиент для YandexGPT API
  - `GigaChatClient` - клиент для GigaChat API
  - `RAGPipeline` - основной RAG pipeline

- `src/config.py` - управление конфигурацией

#### Интерфейсы
- `app.py` - веб-интерфейс Streamlit
- `index_documents.py` - CLI скрипт для индексации документов
- `run.sh` / `run.bat` - скрипты запуска для Linux/Mac и Windows

#### Конфигурация
- `configs/config.yaml` - основной конфигурационный файл
- `.env.example` - шаблон переменных окружения
- `requirements.txt` - зависимости Python

#### Документация
- `README.md` - основная документация
- `USAGE_EXAMPLES.md` - примеры использования
- `CHANGELOG.md` - история изменений

#### Тесты
- `tests/test_parsers.py` - тесты парсеров
- Поддержка pytest

### Технические характеристики

#### Поддерживаемые модели
- Эмбеддинги: paraphrase-multilingual-mpnet-base-v2
- LLM: YandexGPT (yandexgpt, yandexgpt-lite), GigaChat

#### Производительность
- Индексация: ~10-50 документов/минуту
- Поиск: <1 секунда
- Ответ на вопрос: ~3-10 секунд

#### Безопасность
- Поддержка переменных окружения для API ключей
- `.gitignore` для защиты конфиденциальных данных

### Зависимости

#### Основные
- Python 3.8+
- Streamlit 1.29.0
- ChromaDB 0.4.18
- Sentence Transformers 2.2.2
- PyPDF 3.17.1
- python-docx 1.1.0

#### Опциональные
- pytest для тестирования
- camelot-py для улучшенного извлечения таблиц
- tabula-py как альтернатива для таблиц

## Известные проблемы

### Версия 1.0.0
- PDF с комплексной версткой могут парситься неточно
- Таблицы в PDF требуют дополнительных библиотек для качественного извлечения
- GigaChat использует самоподписанные сертификаты (verify=False)
- Большие документы (>100 страниц) могут медленно индексироваться
